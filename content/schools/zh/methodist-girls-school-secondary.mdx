美以美女子中学（Methodist Girls' School，简称 MGS）是新加坡六所自主中学之一。学校创立于 1887 年，以循道卫理宗（Methodist）教育传统为根基，是一所专收女生的单性别学校，校址位于新加坡西部 Blackmore Drive，毗邻阿尔伯特王公园地铁站。

美以美女子中学开设两条学术轨道：综合课程（IP）通往 IB 文凭课程，无需参加 O 水准考试；另设四年制 O 水准课程。作为自主学校，美以美女子中学在教育部全国框架内，拥有比政府学校或政府辅助学校更大的运营灵活度。自主学校学费与一般政府学校有所不同，请参阅[教育部学费页面](https://www.moe.gov.sg/secondary/fees)及美以美女子中学招生页面了解最新收费详情。

在新加坡女校中，美以美女子中学有两大特色：其一，IP 轨道通向 IB 文凭（而非 A 水准）；其二，设有音乐特选课程（MEP）——这是教育部在全国指定的中学音乐发展途径之一。

## 课程与学习计划

### 综合课程（IP · IB 文凭）

美以美女子中学 IP 轨道为六年制。IP 学生在美以美女子中学完成中学阶段后，直接升读 IB 文凭课程，无需参加 O 水准考试。美以美女子中学的 IP 轨道与英华自主中学联办 IB 文凭课程——英华自主中学为本校及联属 IP 学生提供男女合校制 IB 文凭课程。这是新加坡少数以 IB 文凭（而非 A 水准）作为 IP 终点的学校之一。

对于 P6 家长：美以美女子中学的 IP–IB 路径是六年制承诺。若学生在中学阶段退出 IP，将按年级安排转入相应的 O 水准课程。请参阅[美以美女子中学官方网站](https://www.mgs.moe.edu.sg/)了解最新课程架构及升读标准。

### O 水准课程

美以美女子中学提供四年制 O 水准课程，供有意参加全国统一考试的学生选读。IP 及 O 水准两个轨道的学生共用同一校园、课外活动及学校社群。

### 音乐特选课程（MEP）

美以美女子中学开设音乐特选课程（MEP），这是教育部在全国少数中学指定设立的课程。MEP 学生将音乐作为正式学术科目修读，并可通过 DSA-Sec 申请 MEP 才能领域。中学 MEP 是通往初级学院高级音乐课程的途径之一。

如需了解各课程差异，请参阅[什么是 DSA-Sec？](/what-is-dsa)。

## 入学途径

### 小六会考与中一派位

美以美女子中学的 IP 及 O 水准派位成绩截点（COP）各有不同，且每年均可能变动，入学竞争较为激烈。如需了解最新及历年小六会考 COP 数据，请访问[小六会考历年 COP 数据](/psle-cop)。

### 中学直接收生计划（DSA-Sec）

美以美女子中学参与 DSA-Sec，才能领域涵盖体育、表演艺术、语文及学术类。具体才能领域包括：音乐特选课程（MEP）、STEM 与计算思维、英文语文，以及花样游泳、保龄球、足球、篮网球、垒球、壁球、游泳和网球等多项体育项目。

美以美女子中学的 DSA 录取名额视才能领域不同，可能分配至 IP 或 O 水准轨道——申请前请在学校官方 DSA 页面核实各才能领域对应的轨道。

才能领域每年均可能调整。请使用 [DSA 学校搜索工具](/dsa-finder)查看当年最新才能领域。申请关键日期请见 [DSA-Sec 时间表](/timeline)。各才能领域备考指南请见 [DSA 面试准备](/dsa-interview)。历年录取结果请见 [DSA 录取结果](/dsa-results)。

## 常见问题

### 美以美女子中学只收女生吗？

是的。美以美女子中学中学部是单性别女校。IP 轨道的 IB 文凭阶段（最后两年）为男女合校制。

### 我的女儿需要多少小六会考分数才能入读美以美女子中学？

美以美女子中学 IP 及 O 水准派位的成绩截点各有不同，且每年均可能变动，入学竞争较为激烈。如需了解最新及历年数据，请访问[小六会考历年 COP 数据](/psle-cop)。

### 美以美女子中学是否参与 DSA-Sec？

是的。美以美女子中学参与 DSA-Sec，才能领域涵盖体育、音乐（MEP）、语文及学术类。请使用 [DSA 学校搜索工具](/dsa-finder)查看当年最新才能领域，并核实各才能领域对应 IP 或 O 水准轨道。

### 美以美女子中学的 IP 轨道与其他 IP 学校有何不同？

美以美女子中学的 IP 轨道以 IB 文凭课程作为终点，而非大多数新加坡 IP 学校所采用的 A 水准考试。IB 文凭是一项国际认可资质，评估框架与 A 水准不同。家长应深入了解两种路径，判断哪种更适合女儿的学习目标。如需 DSA 及学校选择流程的整体介绍，请参阅[什么是 DSA-Sec？](/what-is-dsa)。

### 什么是美以美女子中学的 MEP，DSA 申请 MEP 如何进行？

音乐特选课程（MEP）是教育部指定课程，学生将音乐作为正式学术科目修读。美以美女子中学 MEP 才能领域的 DSA-Sec 申请，通常须经学校甄选流程，一般包括试奏或面试（或两者兼有）。请在备考前仔细阅读美以美女子中学当年 DSA 招生页面所列的具体要求。备考指南请参阅 [DSA 面试准备](/dsa-interview)。

### 如果女儿未获 DSA 录取，会否影响中一派位？

不会。未获 DSA-Sec 录取，不影响标准中一小六会考派位流程。若孩子的小六会考成绩符合 IP 或 O 水准派位成绩截点，且将美以美女子中学列入志愿，仍可通过中一派位入读。已接受任何学校 DSA-Sec 录取的学生，将退出中一派位统一报名。
