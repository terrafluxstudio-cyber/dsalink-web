Naval Base Secondary School — பொதுவாக NBSS என அழைக்கப்படும் — சிங்கப்பூரின் வடக்குப் பகுதியில் Yishun பகுதியில் Yishun Ring Road-ல் அமைந்த ஒரு அரசு ஆண் மற்றும் பெண் கலப்பு இரண்டாம் நிலைப் பள்ளி. இந்தப் பள்ளி தேசிய பாடத்திட்டத்தை மதிப்பு சார்ந்த குணநலன் வளர்ச்சியுடனும், கலையில் பயன்பாட்டுக் கற்றல் வலியுறுத்தலுடனும் இணைக்கிறது.

Naval Base முழு Subject-Based Banding-உடன் முதன்மை விரைவு மற்றும் சாதாரண கல்வி வழிகளை வழங்குகிறது; IP பள்ளி அல்ல. இசைக்குழு மற்றும் பாடகர் மரபு, காட்சிக் கலை மற்றும் நடன கலாச்சாரம், மற்றும் விளையாட்டின் மூலம் குணநலன் வளர்க்கும் சுறுசுறுப்பான கவனத்திற்காக பெயர் பெற்றுள்ளது.

## திட்டங்கள் மற்றும் பாடத்திட்டம்

### விரைவு மற்றும் சாதாரண கல்வி வழிகள்

Naval Base முழு Subject-Based Banding-உடன் முதன்மை விரைவு O-Level மற்றும் சாதாரண கல்வி வழிகளை வழங்குகிறது, மாணவர்கள் தங்கள் வலிமைகளின் அடிப்படையில் தனிப்பட்ட பாடங்களை உயர் மட்டத்தில் எடுக்க அனுமதிக்கிறது.

P6 குடும்பங்களுக்கு: Naval Base ஒரு முதன்மை பள்ளி, IP பள்ளி அல்ல. S1 இடமாற்றல் கட்டமைப்பின்படி வழி நிர்ணயம் செய்யப்படுகிறது. தற்போதைய கல்வி திட்ட அமைப்பிற்கு [Naval Base அதிகாரப்பூர்வ இணையதளம்](https://www.navalbasesec.moe.edu.sg/) பார்க்கவும்.

### பயன்பாட்டுக் கற்றல் மற்றும் விளையாட்டின் மூலம் குணநலன்

Naval Base கலையில் Applied Learning Programme (ALP) மற்றும் விளையாட்டின் மூலம் குணநலன் கல்வியை மையமாகக் கொண்ட Learning for Life Programme (LLP) நடத்துகிறது. இந்தப் பள்ளி திட்டங்கள் கல்விப் பாடத்திட்டத்திற்கு அப்பால் பள்ளியின் பயன்பாட்டுக் கற்றல் மற்றும் மாணவர் வளர்ச்சிக்கு அடித்தளமாக அமைகின்றன.

DSA-Sec செயல்முறை பற்றிய விளக்கத்திற்கு [DSA-Sec என்றால் என்ன?](/what-is-dsa) காண்க.

## பாட வெளி செயல்பாடுகள்

Naval Base இசை கலை, விளையாட்டு, கிளப்புகள் மற்றும் சீருடை குழுக்களில் CCA நடத்துகிறது.

### இசை கலை

- **பாடகர் குழு (Choir)**
- **Concert Band**
- **இந்திய நடனம்**
- **நவீன நடனம்**

### விளையாட்டு

- **மேசை டென்னிஸ்**
- பள்ளியின் விளையாட்டின் மூலம் குணநலன் கவனத்தை ஆதரிக்கும் பல்வேறு அணி மற்றும் தனிநபர் விளையாட்டுகள்

### காட்சிக் கலை மற்றும் கிளப்புகள்

- **காட்சிக் கலை / கலை கிளப்**
- சமூக சேவை மற்றும் தலைமை குழுக்கள்
- குணநலன் மற்றும் தலைமை வளர்ச்சியை ஆதரிக்கும் சீருடை குழுக்கள்

CCA பெயர்கள் மற்றும் கிடைக்கும் தன்மை மாறலாம். [Naval Base அதிகாரப்பூர்வ இணையதளம்](https://www.navalbasesec.moe.edu.sg/) தற்போதைய CCA பட்டியலுக்கான அதிகார ஆதாரம்.

## வளாகம் மற்றும் வசதிகள்

Naval Base Secondary School சிங்கப்பூரின் வடக்குப் பகுதியில் Yishun பகுதியில் **901 Yishun Ring Road, Singapore 768689** என்ற முகவரியில் அமைந்துள்ளது.

**அருகிலுள்ள MRT நிலையம்:**

- **Yishun** (North South Line NS13) — குறுகிய பேருந்து பயணம்

வளாக வசதிகளில் பொது மற்றும் சிறப்பு வகுப்பறைகள், கலை ஸ்டுடியோக்கள், அறிவியல் ஆய்வகங்கள், பல்நோக்கு மண்டபம், இசை கலை மற்றும் இசை இடங்கள், விளையாட்டு மைதானங்கள் மற்றும் மைதானங்கள் அடங்கும். வசதி விவரங்கள் மற்றும் வளாக வரைபடத்திற்கு [Naval Base அதிகாரப்பூர்வ இணையதளம்](https://www.navalbasesec.moe.edu.sg/) பார்க்கவும்.

## சேர்க்கை வழிகள்

### PSLE மற்றும் S1 Posting

Naval Base விரைவு மற்றும் சாதாரண posting குழுக்களில் S1 posting மூலம் மாணவர்களை சேர்க்கிறது. வழி மற்றும் ஆண்டுக்கேற்ப COP மாறுகிறது. [PSLE COP வரலாற்றுத் தரவு](/psle-cop) காண்க.

### நேரடி பள்ளி சேர்க்கை திட்டம் (DSA-Sec)

Naval Base இசை கலை, காட்சிக் கலை மற்றும் விளையாட்டு திறமை பகுதிகளில் DSA-Sec-ல் பங்கேற்கிறது. இசைக்குழு மற்றும் பாடகர் மரபு, நடனம் மற்றும் காட்சிக் கலை கலாச்சாரம், மற்றும் விளையாட்டின் மூலம் குணநலன் கவனம் கொண்ட ஆண் மற்றும் பெண் கலப்பு பள்ளியாக, Naval Base DSA இந்த வலிமைகளில் விண்ணப்பதாரர்களை ஈர்க்கிறது. DSA சலுகைகள் மாணவர்களை பள்ளியின் முதன்மை வழிகளில் சேர்க்கின்றன.

வழங்கப்படும் திறமை பகுதிகள் ஆண்டுதோறும் மாறலாம். தற்போதைய பட்டியலுக்கு [DSA-Sec பள்ளி தேடல்](/dsa-finder) பயன்படுத்தவும். [DSA-Sec காலவரிசை](/timeline), [DSA தயாரிப்பு வழிகாட்டிகள்](/dsa-interview), மற்றும் [DSA முடிவுகள்](/dsa-results) காண்க.

## அடிக்கடி கேட்கப்படும் கேள்விகள்

### Naval Base Secondary ஆண் மற்றும் பெண் கலப்பு பள்ளியா?

ஆம். Naval Base Secondary School ஒரு ஆண் மற்றும் பெண் கலப்பு பள்ளி.

### Naval Base IP வழங்குகிறதா?

இல்லை. Naval Base முதன்மை விரைவு O-Level, சாதாரண கல்வி (Normal Academic), மற்றும் சாதாரண தொழில்நுட்பம் (Normal Technical) வழிகளை வழங்குகிறது. ஒருங்கிணைந்த திட்டம் (IP) பள்ளி அல்ல.

### என் குழந்தைக்கு Naval Base-க்கு எவ்வளவு PSLE மதிப்பெண் தேவை?

Naval Base விரைவு மற்றும் சாதாரண posting குழுக்களில் போட்டித்தன்மையானது, COP வழி மற்றும் ஆண்டுதோறும் மாறுகிறது. [PSLE COP வரலாற்றுத் தரவு](/psle-cop) காண்க.

### Naval Base DSA-Sec வழங்குகிறதா?

ஆம். Naval Base இசை கலை, காட்சிக் கலை மற்றும் விளையாட்டு திறமை பகுதிகளில் DSA-Sec-ல் பங்கேற்கிறது. தற்போதைய வழங்கல்களை காண [DSA-Sec பள்ளி தேடல்](/dsa-finder) பயன்படுத்தவும்.

### என் குழந்தை Naval Base-ல் DSA இடம் பெறவில்லை என்றால் என்ன ஆகும்?

DSA சலுகை பெறாதது வழக்கமான S1 PSLE posting செயல்முறையை பாதிக்காது. PSLE மதிப்பெண் தகுதியாக இருந்து பள்ளியை விருப்பங்களில் பட்டியலிட்டால் Naval Base-ல் சேரலாம். எந்த பள்ளியிலும் DSA ஏற்றுக்கொண்ட மாணவர்கள் S1 posting-லிருந்து முழுவதுமாக விலகுவார்கள்.
