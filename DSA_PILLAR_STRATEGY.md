# DSALink 内容策略：Pillar + Cluster

> 来源：2026-06-02 session 讨论，从竞品分析得出

---

## 核心发现：成功案例 babagoeschina.com

- **Magda**，波兰妈妈住瑞士，2024年1月建站，1.5年进 Lonely Planet 2025
- 做起来的原因：**一篇极度完整的文章把整个关键词全吃掉**
- 代表作：`/china-with-kids-all-you-need-to-know` — 49分钟阅读量，覆盖所有子话题，每段末尾内链到详情页
- 这叫 **Pillar + Cluster 模型**

---

## Pillar + Cluster 模型说明

```
Pillar页（一篇超完整的总览）
    ↓ 内链到
各个 Cluster 子页（每个话题的深度详情页）
```

**Pillar页的作用：**
- 排名宽泛头部关键词（如 "DSA Singapore guide 2026"）
- Google 看到这页覆盖所有相关话题 → 集中权重
- 用户从这页导航到各子页 → 停留时间拉长

**子页的作用：**
- 各自排名长尾关键词（如 "DSA basketball interview Singapore"）
- 深度覆盖单一话题

## 主页 vs Pillar 页的区别

| | 主页（/） | Pillar 页（/what-is-dsa） |
|--|---------|------------------------|
| 目标读者 | 已知道 DSALink 的人 | Google 搜进来的陌生人 |
| 功能 | 品牌门面、CTA、导航 | SEO 落地、内容深度、内链枢纽 |
| 内容 | 简洁、引流 | 完整、详尽 |

**主页上需要有一个明显入口链接到 Pillar 页。**
